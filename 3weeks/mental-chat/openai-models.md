## 🏷️ 모델별 상세 설명

### 🚀 GPT-5 시리즈 — 최신 플래그십

【 GPT-5.4 Pro 】
- 가장 강력한 모델. 1.1M 토큰 초대형 컨텍스트.
- 고난도 추론, 복잡한 에이전트 작업, 이미지 이해까지.
- 가격이 매우 비싸서 **실서비스 전면 사용은 비추천**. 중요한 판단에만 선택적으로.

【 GPT-5.4 / GPT-5.2 / GPT-5.1 / GPT-5 】
- 일반 용도 최신 세대. 버전 숫자가 높을수록 성능·가격 모두 상승.
- **GPT-5.2**는 코딩/도구 사용에 특화. 개발 워크플로우에 추천.
- **GPT-5.1 / GPT-5**는 범용 채팅/콘텐츠 생성의 표준.

【 GPT-5 Mini 】
- GPT-5 대비 약 **1/5 가격**, 성능 손실은 크지 않음.
- 실사용 API 서비스에 가장 추천. 가성비 최고.

【 GPT-5 Nano 】
- 초저가(입력 70원/1M). 분류, 태깅, 요약 같은 단순 작업 대량 처리용.
- 대화·추론에는 부적합.

---

### 📚 GPT-4.1 시리즈 — 긴 컨텍스트 전용

【 GPT-4.1 】
- **1M 토큰** 컨텍스트. 긴 문서, 대용량 코드베이스 분석에 강함.
- 추론 성능도 o3급. 가격도 o3와 동일.

【 GPT-4.1 Mini 】
- 1M 컨텍스트를 저렴하게 사용하고 싶을 때.
- 긴 문서 RAG 백엔드로 적합.

【 GPT-4.1 Nano 】
- **1M 컨텍스트 + 초저가**의 유일한 조합.
- 대량 로그 분석, 벌크 문서 처리용.

---

### 🎨 GPT-4o 시리즈 — 멀티모달 전문

【 GPT-4o 】
- 텍스트·이미지·음성·영상을 단일 모델에서 처리.
- 음성 대화(Realtime API) 기반. UX 중심 앱에 적합.

【 GPT-4o Mini 】
- 현재 **가장 많이 쓰이는 채팅 기본 모델**.
- 저가 + 준수한 품질 + 빠른 응답. (이 프로젝트의 `mental-chat`이 사용 중)
- 간단한 챗봇, 고객지원, 프로토타입에 표준.

---

### 🧠 o 시리즈 — 추론 특화

【 o3 】
- 복잡한 논리, 수학, 코딩 문제에 강함.
- 답변 전에 **내부적으로 '생각'하는 토큰**을 소모 → 실제 비용은 표시가보다 높음.

【 o4-Mini 】
- o3의 약 **절반 가격**으로 추론 능력 제공.
- 수학/코딩 보조, 에이전트 플래닝에 가성비 우수.

【 o1 】
- 1세대 추론 모델. 가격 대비 성능은 이제 o3/o4-mini가 더 좋음.
- 기존 워크플로우 호환성이 필요할 때만 사용.

## 🎁 비용 절감 팁

1. **Batch API**: 24시간 내 비동기 처리 가능한 작업은 **50% 할인**.
2. **Cached Input**: 같은 시스템 프롬프트를 반복 사용하면 캐시 가격(보통 입력가의 25%)으로 청구.
3. **모델 다운그레이드**: 복잡한 작업만 상위 모델, 나머지는 Mini/Nano로 라우팅.
4. **max_tokens 제한**: 출력이 입력보다 4~8배 비싸므로 출력 길이 제한이 가장 효과적.
5. **스트리밍 + 조기 종료**: 불필요한 출력 토큰 낭비 방지.
