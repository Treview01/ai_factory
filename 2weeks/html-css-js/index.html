시맨틱/구조 태그

article 태그 - 블로그 글 예시

2026년 3월 28일 | 작성자: 홍길동

article 태그는 독립적인 콘텐츠를 나타냅니다. 블로그 글, 뉴스 기사, 포럼 글 등에 사용합니다.

태그: HTML, 시맨틱

aside 태그

aside는 본문과 간접적으로 관련된 부가 정보를 나타냅니다. 사이드바, 광고, 관련 링크 등에 사용합니다.

div 태그

div는 의미 없는 블록 컨테이너입니다. 스타일링이나 스크립트 목적으로 사용합니다.

이것은 span 태그입니다. 인라인 컨테이너로, 텍스트 일부에 스타일을 적용할 때 사용합니다.

details와 summary 태그 (클릭하여 펼치기)

이것은 접을 수 있는 콘텐츠입니다. details 태그 안에 숨겨진 내용이 들어갑니다.

summary는 항상 보이는 제목 부분입니다.

기본 펼침 상태 (open 속성)

open 속성을 추가하면 기본적으로 펼쳐진 상태로 표시됩니다.

dialog 태그

이것은 대화 상자(모달)입니다.

template 태그

template 태그의 내용은 화면에 표시되지 않습니다.

slot 태그 (웹 컴포넌트용)

slot 태그는 웹 컴포넌트의 Shadow DOM 내에서 사용됩니다.
